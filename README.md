# Taylor Swift Eras Guide

Simple guide to Taylor Swift's 12 albums.

## Pages
- `index.html` - Main page with all albums
- `ts1.html` to `ts12.html` - Individual album pages

## Album Pages Include
- Album cover and details
- Description paragraphs
- Track listing
- Next/Previous navigation

## Albums
- Taylor Swift (2006)
- Fearless (2008)
- Speak Now (2010)
- Red (2012)
- 1989 (2014)
- Reputation (2017)
- Lover (2019)
- Folklore (2020)
- Evermore (2020)
- Midnights (2022)
- The Tortured Poets Department (2024)
- The Life of A Showgirl (2025)
